# Arrays, Slices & Maps in Go

These are core data structures in Go.

--------------------------------------------------

# Arrays

Arrays have fixed size.

Syntax:

var numbers [5]int

Example:

numbers := [5]int{1, 2, 3, 4, 5}

--------------------------------------------------

# Accessing Array Elements

Array indexing starts at 0.

Example:

fmt.Println(numbers[0])

--------------------------------------------------

# Length of Array

Use len()

Example:

len(numbers)

--------------------------------------------------

# Looping Through Arrays

Use range keyword.

Example:

for index, value := range numbers {
    fmt.Println(index, value)
}

--------------------------------------------------

# Slices

Slices are dynamic arrays.

Syntax:

scores := []int{90, 80, 70}

--------------------------------------------------

# append()

Adds elements to slices.

Example:

scores = append(scores, 100)

--------------------------------------------------

# Slice Operations

slice[start:end]

Example:

scores[1:3]

--------------------------------------------------

# make()

Creates slices, maps, and channels.

Example:

data := make([]int, 5)

--------------------------------------------------

# copy()

Copies one slice into another.

Example:

copy(destination, source)

--------------------------------------------------

# Multi-dimensional Slices

Example:

matrix := [][]int{
    {1, 2},
    {3, 4},
}

--------------------------------------------------

# Maps

Maps store key-value pairs.

Syntax:

map[keyType]valueType

Example:

students := map[string]int{
    "Alice": 90,
}

--------------------------------------------------

# Accessing Map Values

Example:

students["Alice"]

--------------------------------------------------

# Adding Values

Example:

students["Bob"] = 85

--------------------------------------------------

# Deleting Values

Use delete()

Example:

delete(students, "Bob")

--------------------------------------------------

# Checking if Key Exists

Example:

value, exists := students["Alice"]

--------------------------------------------------

# Looping Through Maps

Example:

for key, value := range students {
    fmt.Println(key, value)
}

--------------------------------------------------

# Sets in Go

Go has no built-in set type.

Maps with bool values simulate sets.

Example:

set := map[string]bool{
    "Go": true,
}

--------------------------------------------------

# Important Notes

1. Arrays are fixed size.

2. Slices are used more commonly than arrays.

3. Maps are unordered.

4. Slices internally contain:
    pointer
    length
    capacity

--------------------------------------------------

# Python vs Go

Python List:
    numbers = [1, 2, 3]

Go Slice:
    numbers := []int{1, 2, 3}

--------------------------------------------------

Python Dictionary:
    {"a": 1}

Go Map:
    map[string]int{
        "a": 1,
    }

--------------------------------------------------

Python Set:
    {1, 2, 3}

Go:
    map[int]bool
